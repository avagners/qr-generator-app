# QR Code Generator

## Описание
Простое одностраничное веб-приложение (SPA) для генерации QR-кодов прямо в браузере. Приложение работает полностью на стороне клиента без необходимости в сервере или передаче данных на внешние сервисы.

## Особенности
- Генерация QR-кодов из текста, URL, контактной информации и других данных
- Работа полностью в браузере (без отправки данных на сервер)
- Возможность настройки размера и цвета QR-кода
- Экспорт QR-кода в формате PNG или SVG
- Адаптивный дизайн для использования на мобильных устройствах и компьютерах
- Быстрая и простая генерация без регистрации или установки

## Технологии
- **HTML5** - структура страницы
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (ES6+)** - основная логика приложения
- **qrcode.js** - библиотека для генерации QR-кодов
- **Webpack** - сборка проекта (опционально)
- **Vite** - быстрая разработка и сборка (опционально)

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите в режиме разработки: `npm run dev`
4. Соберите проект: `npm run build`

## Использование
1. Введите текст или URL в поле ввода
2. Настройте параметры QR-кода (размер, цвет)
3. Нажмите кнопку "Сгенерировать"
4. Сохраните полученный QR-код

## Безопасность
Поскольку всё приложение работает в браузере, ваши данные не покидают ваше устройство. Это обеспечивает максимальную конфиденциальность и безопасность.

## Совместимость
Приложение поддерживает все современные браузеры (Chrome, Firefox, Safari, Edge).
